Nano ESP32_SSD1306_OLED_Example
Simple implementation for an 0.96" I2C OLED Display with a Nano ESP32

1. Download the AcroboticSSD1306 Library in Arduino IDE.
2. Download and compile ESP32-OLED.ino



| OLED       | Nano ESP32         |
| ------------- |:-------------:|
| VCC | 3V3 |
| GND | GND |
| SCL | GPIO22 - I2C-SCL |
| SDA | GPIO21 - I2C-SDA |
